# 📚 IGCSE Revision Hub

A premium, browser-based study vault for Cambridge IGCSE students. Upload PDFs, create flashcards, organise past papers and manage access codes — all stored locally in your browser via IndexedDB.

---

## 🗂 Project Structure

```
igcse-revision-hub/
├── index.html              # The full static website
├── app.py                  # Streamlit wrapper to serve index.html
├── requirements.txt        # Python dependencies
├── igcse-logo-cropped.png  # Logo image (add your own — see note below)
└── README.md
```

> ⚠️ **Logo note:** The site references `igcse-logo-cropped.png` in the same folder. Make sure this file is present in the repository root before deploying. If you don't have it, remove the `<img>` tag in `index.html` and replace it with a plain text brand name.

## 🚀 Option 1 — Host on GitHub Pages (Recommended for the static site)

GitHub Pages serves static HTML directly — no Python needed.

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under *Source*, select `Deploy from a branch` → `main` → `/ (root)`.
4. Click **Save**. Your site will be live at:
   ```
   https://<your-username>.github.io/<repo-name>/
   ```

That's it. All features (IndexedDB, file uploads, flashcards) work natively in the browser.

## 🚀 Option 2 — Deploy on Streamlit Community Cloud

Use this if you specifically want the Streamlit platform.

### Step 1 — Push to GitHub

```bash
git init
git add .
git commit -m "Initial commit — IGCSE Revision Hub"
git branch -M main
git remote add origin https://github.com/<your-username>/<repo-name>.git
git push -u origin main
```

### Step 2 — Deploy on Streamlit Cloud

1. Go to [share.streamlit.io](https://share.streamlit.io) and sign in with GitHub.
2. Click **New app**.
3. Select your repository, branch (`main`), and set the **Main file path** to `app.py`.
4. Click **Deploy**.

Your app will be live at:
```
https://<your-username>-<repo-name>-app-<hash>.streamlit.app
```

## 💻 Run Locally

```bash
# Install dependencies
pip install -r requirements.txt

# Run the Streamlit app
streamlit run app.py
```

Or open `index.html` directly in any browser — no server needed for the static version.

## 🔐 Owner PIN

The default owner PIN is `1234`. To change it, open `index.html` and find this line near the top of the `<script>` block:

```js
const OWNER_PIN="1234"
```

Replace `1234` with your chosen PIN before deploying.
